# gomoku

双人对战的五子棋

## 说明书
游戏中, 方向键移动光标, O 按钮在当前光标处落子, X 按钮重置当前游戏

游戏结束后, O 按钮重新开始下一局游戏

黑棋与白棋玩家使用同一个光标, 轮流落子. 当一个玩家在行, 列或斜线连成五颗棋子时, 游戏结束, 该玩家胜利

游戏不会判定平局, 如果无法继续落子, 或两方都不可能连成五颗子, 需要玩家手动重置游戏
